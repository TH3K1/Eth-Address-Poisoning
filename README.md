
# Address Poisoner - Eth- Sol - Bnb

This is a simple address poisoning script i made as a showcase for my portfolio, the usage is very simple yet very effective, it supports all the major blockchains including all the EVM ones and Solana.



## Installation

The only requisite is Python's latest version and the following libraries installed:

```bash
pip install web3==7.10.0
pip install python-dotenv==1.1.0
pip install py-solc-x==2.0.3
```
    
## Features

- Multi Poison Attack for a faster and more efficient way to manage your poisoned wallets.
- Live previews of the targeted wallets.
- Multi chain support, evm and sol.
